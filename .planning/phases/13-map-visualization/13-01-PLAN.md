**Create `lib/types.ts`:**
- `MapLayer` interface: `{ id: string, filename: string, geojson: GeoJSON.FeatureCollection, color: string, visible: boolean, featureCount: number }`
- `TILE_LAYERS` const object with three entries:
  - `osm`: OpenStreetMap (`https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png`)
  - `satellite`: ESRI World Imagery (`https://server.arcgisonline.com/ArcGIS/rest/services/World_Imagery/MapServer/tile/{z}/{y}/{x}`)
  - `topo`: OpenTopoMap (`https://{s}.tile.opentopomap.org/{z}/{x}/{y}.png`)
- Each tile layer entry has: `name`, `url`, `attribution`
- Export `TileLayerKey = keyof typeof TILE_LAYERS`

**Create `lib/parse-spatial-file.ts`:**
- Export `parseSpatialFile(file: File): Promise<GeoJSON.FeatureCollection>`
- Route by file extension (`.geojson`/`.json`, `.kml`, `.kmz`, `.zip`)
- GeoJSON: `file.text()` then `JSON.parse()`
- KML: `file.text()`, `new DOMParser().parseFromString(text, "text/xml")`, `toGeoJSON.kml(dom)`
- KMZ: `file.arrayBuffer()`, `JSZip.loadAsync(buffer)`, find `.kml` file in zip, parse as KML
- Shapefile ZIP: `file.arrayBuffer()`, `shp(buffer)`. Handle array result from shpjs (if `Array.isArray(result)`, use `result[0]`)
- Throw descriptive error for unsupported extensions
- Ensure every feature has an `id` property (generate incrementing IDs if missing)

**Create `lib/layer-colors.ts`:**
- Export `LAYER_COLORS` array of 10 distinct colors: `["#3B82F6", "#EF4444", "#10B981", "#F59E0B", "#8B5CF6", "#EC4899", "#06B6D4", "#F97316", "#6366F1", "#14B8A6"]`
- Export `getLayerColor(index: number): string` that cycles through the array

**Create `lib/session-store.ts`:**
- Export `saveLayers(layers: MapLayer[]): void` -- serialize to sessionStorage key `"map-visualizer-layers"`. Wrap in try/catch for QuotaExceededError.
- Export `loadLayers(): MapLayer[] | null` -- deserialize from sessionStorage. Return null on error or missing data.
- Export `clearLayers(): void` -- remove the key.

**Create `src/app/(public)/tools/visualize/map-visualizer.tsx`:**
"use client" component. This is the main orchestrator:
- State: `layers: MapLayer[]`, `activeLayerId: string | null`, `baseMap: TileLayerKey` (default "osm")
- On mount: load layers from sessionStorage via `loadLayers()`
- On layers change: save to sessionStorage via `saveLayers(layers)`
- `handleFileParsed(file: File, geojson: FeatureCollection)`: create MapLayer with `crypto.randomUUID()` id, assign color via `getLayerColor(layers.length)`, add to layers state, set as active
- `handleToggleVisibility(layerId: string)`: toggle `visible` field
- `handleColorChange(layerId: string, color: string)`: update layer color
- `handleRemoveLayer(layerId: string)`: remove from layers, clear activeLayerId if it was active
- `handleSelectLayer(layerId: string)`: set activeLayerId
- `handleBaseMapChange(key: TileLayerKey)`: update baseMap state
- Layout: Full viewport height (`h-screen` or `h-[calc(100vh)]`), relative positioning. The map fills the entire area. Floating panels positioned absolute:
  - Top-left: Upload panel (collapsible)
  - Left side below upload: Layer panel (collapsible)
  - A "zoom to fit" button (top-right area, small floating button with crosshair/target icon)
- Use `next/dynamic` to import `LeafletMap` with `ssr: false` and a loading skeleton ("Loading map...")
- Pass layers, baseMap, activeLayerId, and a `fitBoundsKey` (increment on new layer add or zoom-to-fit click) to the map component
- Add a back link/button to `/dashboard` or just the app logo linking to `/` for navigation

  - `pointToLayer` function: return `L.circleMarker(latlng, { radius: 6, color: layer.color, fillColor: layer.color, fillOpacity: 0.7 })` for points
  - `onEachFeature` function: bind hover tooltip (show `feature.properties.name || feature.properties.Name || feature.properties.description || "Feature " + feature.id`) and click popup (show all properties as key-value list in a scrollable div, max-height 200px)
- `CoordinateDisplay` sub-component (inside this file since it uses `useMapEvents`):
  - Track mouse position with `useMapEvents({ mousemove })`, display lat/lng in a fixed position overlay at bottom-left
  - Format: `Lat: XX.XXXXXX, Lng: XX.XXXXXX` in monospace font
  - Render as Leaflet control position (`leaflet-bottom leaflet-left`) with white/90 backdrop
- `FitBounds` sub-component (inside this file since it uses `useMap`):
  - Accept `layers` and `fitKey` props
  - When `fitKey` changes, compute bounds from all visible layers using `L.geoJSON(layer.geojson).getBounds()`, extend a combined bounds, call `map.fitBounds(bounds, { padding: [50, 50] })`
  - Only fit if bounds are valid
